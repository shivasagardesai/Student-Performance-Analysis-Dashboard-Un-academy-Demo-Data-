# Student-Performance-Analysis-Dashboard-Un-academy-Demo-Data-
I created a student performance dashboard using demo data similar to Unacademy. I analyzed batch-wise and test-wise performance, categorized students into top, average, and poor performers, and built visual dashboards using Excel to make insights easy to understand


## 🔹 Objectives
- Perform Batch-wise analysis of students
- Perform Test-wise analysis (Minor & Major tests)
- Identify top, average, and poor performers
- Build an interactive dashboard for insights

## 🔹 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization (Charts & Dashboard)

## 🔹 Analysis Performed

### 📌 Batch-wise Analysis
- Top Performers
- Average Performers
- Poor Performers

### 📌 Test-wise Analysis
- Minor Tests Performance
- Major Tests Performance

## 🔹 Dashboard Features
- Clear categorization of students based on performance
- Visual charts for better understanding
- Easy-to-read insights for decision making

## 🔹 Key Insights
- Identified high-performing students across batches
- Found performance differences between minor and major tests
- Highlighted students needing improvement

## 🔹 Conclusion
This dashboard helps educators track student performance effectively and make data-driven decisions.

## 🔹 Project Output
Excel Dashboard with visual representations of student performance
